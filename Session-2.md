# if와 else

```
if (참 또는 거짓의 조건문)
{
	*참 일시, 실행*
} else if (참 또는 거짓의 조건문)
{
	*참 일시, 실행*
} else {
	*위 조건문에 대하여 참이 아닌경우  실행*
}
```

# switch

```
switch (choice)
{
    case 0:
        Console.WriteLine("가위입니다.");
        break;
    case 1:
        Console.WriteLine("바위입니다.");
        break;
    case 2:
        Console.WriteLine("보입니다.");
        break;
    case 3:
        Console.WriteLine("치트키입니다.");
        break;
    default:
        Console.WriteLine("다 실패 했습니다.");
        break;
}
```

#  break, continue

break는 반복문을 중단시킴
불필요한 계산 최소화

```
for (int i = 2; i < num; i++)
{
    if ((num % i) == 0)
    {
        isPrime = false;
        break;
    }
}
```

## continue

continue; 계속 진행
불필요한 계산 최소화

```
for (int i = 1; i <= 100; i++)
{
    if ((i % 3) != 0)
    {
        continue;
    }
    Console.WriteLine("3의 배수 발견! : {0}", i);
}
```
